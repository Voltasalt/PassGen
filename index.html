PassGen A tiny password manager

About

PassGen is a very very small password manager. All you have to do is enter your master password in the top field, and a site name in the bottom. PassGen will automatically create a password for that site. Check the checkbox to show the password. The password will always contain uppercase, lowercase, digit and special characters, and will always be exactly 57 characters long, ending in "==". PassGen is deterministic, which means you can input your master password and the site name on any computer, anywhere (even offline), and the password will be the same.

Nothing you enter here is ever sent to a server, which means you can download this page onto a USB stick and use it offline, if you want to. You can also check the source code on GitHub if you don't believe me.
